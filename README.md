# TechFeud

TechFeud is an interactive, fast-paced web quiz game inspired by the classic Family Feud format. It focuses on clean, responsive gameplay and smooth interactions so you can run friendly trivia matches locally or deploy them for small groups.

Overview
- Play rounds of rapid-response trivia with configurable question sets.
- Responsive UI with smooth animations and keyboard-friendly controls.
- Designed for quick local development and simple static deployments.

Quick Start
Prerequisites: Node.js (LTS recommended) and npm or yarn.

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Configuration
- Environment and build settings can be adjusted via standard package scripts and environment variables.

Development Notes
- Fast iterative development with hot-reload for UI changes.
- Keep question content in a simple JSON format for easy editing and import.
